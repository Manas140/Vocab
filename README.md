<h1 align="center">.Vocab</h1>
<p align="center">An online word database with the aim of making learning new vocabulary easier</p>

### Technologies used
- NextJs
- Tailwind
- Vercel KV

### Setup

- Clone
```
git clone https://github.com/manas140/Vocab.git
cd Vocab
```

- Install
```
yarn install
```

- Run
```
yarn dev
```
