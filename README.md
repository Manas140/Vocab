# .Vocab

Finding words made pretty, and easy.

### Technologies & APIs

- NextJs
- Tailwind <br/><br/>
- https://www.datamuse.com/api/
- https://dictionaryapi.dev/
- https://random-word-api.vercel.app/

### Setup

- Clone

```
git clone https://github.com/manas140/Vocab.git
cd Vocab
```

- Install

```
bun install
```

- Run

```
bun run dev
```
